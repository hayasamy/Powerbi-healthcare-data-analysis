## 📌 Project Overview

This project presents an **interactive Healthcare Analytics Dashboard built using Power BI** to analyze hospital data, patient demographics, medical conditions, and billing performance.

---

# 🎯 Project Objectives

The main objectives of this dashboard are to:

* Analyze **patient demographics**
* Identify the **most common medical conditions**
* Track **admission patterns**
* Monitor **healthcare billing trends**
* Provide **interactive filters for deeper analysis**

---

# 📊 Key Performance Indicators (KPIs)

The dashboard highlights the following key metrics:

| KPI                        | Description                                           |
| -------------------------- | ----------------------------------------------------- |
| **Total Patients**         | Total number of patients in the dataset               |
| **Total Billing Amount**   | Total revenue generated from treatments               |
| **Average Length of Stay** | Average number of days patients stay in hospital      |
| **Most Common Condition**  | Medical condition with the highest number of patients |

---

# 📄 Dashboard Pages

## 1️⃣ Overview Page

This page provides a **high-level summary of hospital performance and patient distribution**.

### Visualizations

* Patients by Gender
* Admitted Patients by Year
* Patients by Hospital
* Medical Conditions by Hospital (Matrix)

### Questions Answered

* Overall patient distribution
* Hospital patient volume
* Trends in patient admissions
* Distribution of medical conditions across hospitals

---

## 2️⃣ Demographics Page

This page focuses on **patient demographic characteristics and admission patterns**.

### Visualizations
* Patients by Age Group
* Patients by Admission Type
* Patients by Medical Condition
* Patients by Blood Type

### Questions Answered
* Which age groups have the most patients?
* What admission types are most common?
* How do medical conditions differ across genders?
* What is the distribution of patients across blood types?

---

## 3️⃣ Insights Page

This page focuses on **financial and treatment insights**.

### Visualizations
* Total Billing by Insurance Provider
* Total Billing by Medical Condition
* Medication Usage by Condition
* Total Billing Amount by Year

### Questions Answered
* Which insurance providers contribute the most to the billing
* Which conditions generate the highest costs
* Medication distribution by condition
* Billing trends over time

---

### Available Filters

* **Year**
* **Month**
* **Quarter**
* **Medical Condition**
*  **Blood Type**
